# 换皮原则

## 术语替换规则

1. "修为" → "等级"
2. "领主境界达到xx" → "领主达到xx级"
3. "修为达到xx" → "达到xx级"
4. 境界名称（如炼气1阶、出窍5阶等）→ 对应的等级数字（train表的id）
